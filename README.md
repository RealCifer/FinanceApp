# Personal Finance Dashboard

A responsive personal finance dashboard designed to help young professionals track, understand, and improve their financial habits through a clean and intuitive interface.

---

## Overview

This project focuses on simplifying financial management by presenting key information in a structured and visually clear manner. It emphasizes usability, responsiveness, and meaningful data representation.

---

## Problem Statement

Many users find financial tools overwhelming due to complexity and poor information design.

This application provides:
- Clear financial overview  
- Spending insights  
- Goal tracking  
- Financial health evaluation  

---

## Features

Dashboard
- Total balance, income, and expenses
- Monthly spending visualization
- Quick insights

Transactions
- Transaction listing
- Category-based structure
- Search functionality

Insights
- Income vs expense trends
- Weekly analytics
- Behavioral insights

Goals
- Multiple savings goals
- Progress tracking
- Visual indicators

Financial Health
- Health score system
- Spending and savings analysis
- Improvement suggestions

---

## Unique Feature

Financial Health Score

A simplified system that evaluates:
- Spending habits
- Savings consistency
- Financial stability

---
